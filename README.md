# TOTP Generator (Self-made Auth App)

## 概要

本プロジェクトは、TOTP（Time-based One-Time Password）アルゴリズムに基づいて6桁のワンタイムパスワードを生成する、ブラウザ上で動作する簡易認証アプリです。

Google Authenticatorなどの認証アプリと互換性のあるコードを生成できます。

ただし、セキュリティ的に「ちゃんとした認証アプリの代替」を名乗るにはやや勇気が必要です。

---

## 特徴

* ブラウザ完結（インストール不要）
* Base32形式のシークレット対応
* 30秒ごとのTOTP自動更新
* コピー機能付き
* 軽量・依存ライブラリなし（人間のミスだけが依存先）

---

## 技術仕様

* アルゴリズム: TOTP (RFC 6238)
* HMAC: SHA-1
* 時間ステップ: 30秒（変更可能）
* 桁数: 6桁（変更可能）
* 実行環境: モダンブラウザ（Web Crypto API必須）

---

## 使い方

1. シークレットキー（Base32）を入力
2. 表示された6桁コードをログイン画面に入力
3. 30秒ごとに更新されるコードを見て人生の無常を感じる

---

## 動作環境

* Chrome / Edge / Firefox / Safari（最新）
* HTTPS環境推奨（Clipboard APIの都合）

---

## セキュリティ注意事項（重要）

このアプリはTOTPを正しく実装しているため、理論上は認証アプリとして機能します。

しかし以下の点に注意してください：

* シークレットキーはブラウザ上に存在します
* ページ改ざん・拡張機能・マルウェアの影響を受ける可能性があります
* 端末分離が弱いため、スマホ認証アプリより安全性は低いです

つまり：

> 「動く」ことと「安全に運用できる」は別問題です

## 設計思想

* できるだけシンプルに

* 人間はだいたいバックアップを忘れる
